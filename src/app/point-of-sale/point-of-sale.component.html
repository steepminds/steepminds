Point Of Sale With TouchUI

Pointofsale is a software for retailer to carry out its day to day transaction very easily and effectively. It is developed using .NET, using WPF, C# and Entity Framework (SQL SERVER). UI is designed particulary for touchable screen and easy usage.

Features

  • Easy Usage
  • Dual Screen display (For customer and retailer)
  • Monthly, Daily Reports
  • Easily customizable
  • UI dsign especially for touch screen
  • Security and Privacy
